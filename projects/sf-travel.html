About the Client

San Francisco Travel is an association that help people arrange their visits to San Francisco by curating the most interesting ideas, events, attractions and more happening in the City by the Bay.

Every year, Teak, a creative design studio handling this account, helps San Francisco Travel create a report on how the city is progressing. As Teak's summer intern, I took it upon myself to create my own intern project, turning this print report into a website.

Project Info

Languages: HTML5, CSS3, and Javascript

Plugins/Libraries: jQuery, Waypoints,Parallax

Handlers: Sass, Github

My Role

The website is hand-coded by me and is still a work in progress. I am currently working with Teak's designer Angela May Chen to create beautiful layouts for the website.

Please visit the Sf Travel Annual Report 2016 to see the full project live.

Thank you!
